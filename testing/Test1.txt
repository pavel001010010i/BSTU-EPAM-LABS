Аренда автомобилей sixt.com 


1. Поиск автомобиля в недоуступных для компании странах.
Шаги: заходим на главную страницу, в поле pick-up & return, ставим галочку RETURN AT PICK-UP LOCATION,
ищем страну, например Indonesia, отстальные поля заполнять не обязатеьно, жмем enter.
Ожидаемый результат: вывод сообщения 'Sorry, but there are no SIXT stations available near Indonesia!' 
либо подсказки с ближайшими странами/городами, в которых есть автомобили данной компании.

2. Поиск такси на будущее (более 2 месяцев)
Шаги: заходим на сайт, выбираем вкладку RIDE, в контекстном меню находим столбец RIDE SERVICES,
выбираем taxi, далее заполняем  pickup at (откуда) и destination (пункт назначения),
выбираем pickup date/time (дату и время) 08/15/2020 - 09:20.
Ожидаемый результат : отображение поиска либо сообщения 'Offer calculator was unable to calculate an offer'. 

3. Аренда машины задним числом.
Шаги: заходим на главную (rent), заполняем поля pick-up & return, pick-up time, return date, 
return time и в поле pick-up date выбираем прошлое дату (14.09.2019).
Ожидаемый результат: не возможно выбрать дату, так как эти числа заблокированы.

4. Найти атомобиль с количеством мест, большее 7.
Шаги: заходим на главную, заполняем поля pick-up & return, pick-up date, return date, time,
жмем show offers, в поле NO. OF SEATS пробуем выбрать больше 7 мест.
Ожидаемый результат: максимально допустимое значение равно 7.

5. Найти атомобиль с количеством мест, меньше 2 .
Шаги: заходим на главную, заполняем поля pick-up & return, pick-up date, return date, time,
жмем show offers, в поле NO. OF SEATS  пробуем выбрать меньше 2 мест.
Ожидаемый результат: в поле минмально допустимое значение равно 2.

6. Указать возраст водителя меньше 18 лет.
Шаги: заходим на главную, заполняем поля pick-up & return, pick-up date, return date, time,
жмем show offers, в поле DRIVER AGE пробуем выбрать значение меньше 18
Ожидаемый результат: минимальное допустимое значение 18 лет ( в моем случае 21 год).

7. Предоставение подтверждения аренды автомобиля (скачиваемый файл).
Шаги: заходим на главную, заполняем поля pick-up & return, pick-up date, return date, time;
Клацаем show offers, выбираем авто, ознакомившишь с авто жмем select;
Если надо выбираем дополнительные функции, далее continue;
Зполняем поля Street Address, Country Code, Mobile phone number, Country,
если есть что-то из бонусов - Bonus Program -> Number и наконец-то жмем BOOK NOW.
Ожидаемый результат: на странице с информацией о заказе будет ссылка на скачку файла в .pdf .

8. Просмотр истории арендуемых автомобилей.
Шаги: логинимся (log in - user1.user2@gmail.com/qwerty123), заходим на главную, 
заполняем поля pick-up & return, pick-up date, return date, time;
Клацаем show offers, выбираем авто, ознакомившишь с авто жмем select;
Если надо выбираем дополнительные функции, далее continue;
Зполняем поля street address,country code, mobile phone number, country,
если есть что-то из бонусов - Bonus Program -> Number и наконец-то жмем BOOK NOW.
Далее жмем на жмем на Account; В разделе RESERVATIONS видим наш заказы.
Ожидаемый результат: список арендованных авто. 

9. Отмена заказа с подтверждением.
Шаги: заходим на сайт, жмем Log in (логинимся под своей учетной записью), жмем на Account;
В разделе RESERVATIONS видим наш заказ, выбираем, находим кнопку CANEL RESERVATION.
Ожидаемый результат: подтверждение отмены с помощью меседж бокса (yes/no).

10. Добавление кредитной карточки.
Шаги: заходим на сайт, жмем Log in (логинимся под своей учетной записью), жмем на Account;
В разделе PROFILES выбираем PAYMENT METHOD -> ADD NEW PAYMENT METHOD
Заполняем поля Carholder, Car number, Month, Year, CVV и жмем CONFIRM.
Ожидаемый результат: подтверждение карточки либо ERROR OCCURRED.
