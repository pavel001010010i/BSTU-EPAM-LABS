Аренда автомобилей sixt.com 


1. Поиск автомобиля в недоуступных для компании странах.+++
Шаги: заходим на главную страницу, в поле pick-up & return, ставим галочку RETURN AT PICK-UP LOCATION,
ищем страну, например Indonesia, отстальные поля заполнять не обязатеьно, жмем enter.
Ожидаемый результат: вывод сообщения 'Sorry, but there are no SIXT stations available near Indonesia!' 
либо подсказки с ближайшими странами/городами, в которых есть автомобили данной компании.

2. Аренда машины задним числом.++
Шаги: заходим на главную (rent), заполняем поля pick-up & return, pick-up time, return date, 
return time и в поле pick-up date выбираем прошлое дату (14.09.2019).
Ожидаемый результат: не возможно выбрать дату, так как эти числа заблокированы.

3. Найти только гарантированые модели.++
Шаги: заходим на главную, заполняем поля pick-up & return, pick-up date, return date, time,
жмем show offers, в поле Vehicle Type выбираем Guaranteed models only.
Ожидаемый результат: предожение вариантов машин или вывод сообщения ошибки No offers were found for these filter settings.

4. Найти атомобиль с количеством мест, меньше 2 .++
Шаги: заходим на главную, заполняем поля pick-up & return, pick-up date, return date, time,
жмем show offers, в поле NO. OF SEATS  пробуем выбрать меньше 2 мест.
Ожидаемый результат: в поле минмально допустимое значение равно 2.

5. Указать возраст водителя меньше 18 лет.++
Шаги: заходим на главную, заполняем поля pick-up & return, pick-up date, return date, time,
жмем show offers, в поле DRIVER AGE пробуем выбрать значение меньше 18
Ожидаемый результат: минимальное допустимое значение 18 лет ( в моем случае 21 год).

6. Проверка на изменение пароля.++
Шаги: заходим на сайт, жмем Log in (логинимся под своей учетной записью), жмем на Account;
Вкладка Account -> change password, вводим неверный пароль и новый пароль, -change click
Ожидание: вывод сообщения Your current password is wrong.

7. Отмена заказа с подтверждением.++
Шаги: заходим на сайт, жмем Log in (логинимся под своей учетной записью), жмем на Account;
В разделе RESERVATIONS видим наш заказ, выбираем, находим кнопку CANEL RESERVATION.
Ожидаемый результат: подтверждение отмены с помощью меседж бокса (yes/no).

8. Добавление нового профиля.++
Шаги: заходим на сайт, жмем Log in (логинимся под своей учетной записью), жмем на Account;
В разделе PROFILES выбираем ADD PROFILE -> private profile
Заполняем поля Profile Name и Email. Жмем кнопку ADD PROFILE
Ожидаемый результат: вывод сообщения PROFILE CREATED SUCCESSFULLY или неправильная валидация полей.

9. Выбор дополнительного водителя ++
Шаги: логинимся (log in - user1.user2@gmail.com/qwerty123), заходим на главную, 
заполняем поля pick-up & return, pick-up date, return date, time;
Клацаем show offers, выбираю автомобить из предложенных вариантов,
выбираю способ оплаты PAY ONLINE и жму Select -> Continue
Жму в поле ADDITIONAL DRIVER, выбираю количество водителей и жму кновку
ADD.
Ожидание: Водитель добавился или сообщение о недоступности водителя

10. Поиск официального представителя в стране.++
Шаги: логинимся (log in - user1.user2@gmail.com/qwerty123), заходим на главную, 
спускаемся вниз к разделу SIXT LOCATIONS WORLDWIDE, жмем кнопку FIND A LOCATION
далее выбираем континент EUROPE - Albania,  в поисковике ищем город Dhermi
Далее жмем SEE FULL LIST OF LOCATIONS.
Ожидание: варианты представителей.
